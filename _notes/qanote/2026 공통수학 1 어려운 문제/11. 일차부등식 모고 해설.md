## 6번 문제

### 문제

6. 연립부등식
    
    $\begin{cases} 3x \ge x - 3 \cr 2x + 1 \le 11 \end{cases}$
    
    을 만족시키는 모든 정수 $x$의 값의 합은? [3점]
    

① $10$

② $11$

③ $12$

④ $13$

⑤ $14$

### 풀이

첫 번째 부등식을 풀면 다음과 같습니다.

$3x \ge x - 3$

$2x \ge -3$

$x \ge -\frac{3}{2} \quad \cdots \text{ (1)}$

두 번째 부등식을 풀면 다음과 같습니다.

$2x + 1 \le 11$

$2x \le 10$

$x \le 5 \quad \cdots \text{ (2)}$

(1)과 (2)의 공통 범위를 구하면 다음과 같습니다.

$-\frac{3}{2} \le x \le 5$

이 범위를 만족시키는 정수 $x$는 $-1, 0, 1, 2, 3, 4, 5$입니다.

따라서 모든 정수 $x$의 값의 합은 다음과 같습니다.

$(-1) + 0 + 1 + 2 + 3 + 4 + 5 = 14$

**정답: ⑤**

## 18번 문제

### 문제

18. $x$에 대한 연립부등식
    
    $\begin{cases} \lvert ax - 1\rvert < 21 \cr 2x + 3 > 5 \end{cases}$
    
    를 만족시키는 자연수 $x$의 개수가 $2$일 때, 모든 정수 $a$의 값의 합은? [4점]
    

① $1$

② $2$

③ $3$

④ $4$

⑤ $5$

### 풀이

두 번째 부등식을 먼저 정리하여 $x$의 기본 범위를 구합니다.

$2x + 3 > 5$

$2x > 2$

$x > 1 \quad \cdots \text{ (1)}$

연립부등식을 만족시키는 **자연수 $x$의 개수가 2개**이므로, (1)의 조건($x > 1$)에 의해 만족하는 자연수 $x$는 반드시 $2$와 $3$이어야 합니다. 즉, 자연수 $4$는 조건을 만족하지 않아야 합니다.

이제 첫 번째 부등식 $|ax - 1| < 21$에 자연수 $x = 2, 3$을 대입했을 때 성립하고, $x = 4$일 때는 성립하지 않는 정수 $a$의 조건을 찾습니다.

부등식 $|ax - 1| < 21$은 다음과 같이 풀어집니다.

$-21 < ax - 1 < 21$

$-20 < ax < 22 \quad \cdots \text{ (2)}$

**1) $x = 2$가 포함되어야 하므로:**

$-20 < 2a < 22 \implies -10 < a < 11 \quad \cdots \text{ (A)}$

**2) $x = 3$이 포함되어야 하므로:**

$-20 < 3a < 22 \implies -\frac{20}{3} < a < \frac{22}{3} \quad \cdots \text{ (B)}$

**3) $x = 4$는 포함되지 않아야 하므로:**

$4a \le -20$ 또는 $4a \ge 22$ 이어야 합니다. 이를 정리하면,

$a \le -5 \text{ 또는 } a \ge \frac{11}{2} \quad \cdots \text{ (C)}$

**정수 $a$ 구하기:**

(A)와 (B)의 공통 범위를 만족하는 정수 $a$의 후보는 다음과 같습니다.

$-\frac{20}{3} < a < \frac{22}{3} \implies -6.66\dots < a < 7.33\dots$

따라서 가능한 정수는 $-6, -5, -4, \dots, 6, 7$ 입니다.

여기에 조건 (C)를 결합하여 범위를 만족하는 정수를 필터링합니다.

- **케이스 1 ($a \le -5$ 일 때):**
    
    $-\frac{20}{3} < a \le -5$ 를 만족하는 정수 $a$는 **$-6, -5$** 입니다.
    
- **케이스 2 ($a \ge \frac{11}{2}$ 일 때, 즉 $a \ge 5.5$):**
    
    $5.5 \le a < \frac{22}{3}$ 를 만족하는 정수 $a$는 **$6, 7$** 입니다.
    

따라서 가능한 모든 정수 $a$는 **$-6, -5, 6, 7$** 입니다.

모든 정수 $a$의 값의 합을 구하면 다음과 같습니다.

$(-6) + (-5) + 6 + 7 = 2$

**정답: ②**

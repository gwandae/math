### [3쪽 문제 풀이]

#### 1. 두 다항식 $A=2x^2+xy+y^2$, $B=x^2+2xy-y^2$ 에 대하여 $A+B$를 간단히 하면?

- **정답:** $3x^2+3xy$
    
- **풀이:**
    
    동류항끼리 묶어서 결합합니다.
    
    $A+B = (2x^2 + x^2) + (xy + 2xy) + (y^2 - y^2)$
    
    $A+B = 3x^2 + 3xy$
    

#### 2. [2025학년도(24년 시행) 9월] 1. 두 다항식 $A=x^2+3xy+2y^2$, $B=2x^2-3xy-y^2$ 에 대하여 $A+B$를 간단히 하면?

- **정답:** $3x^2+y^2$
    
- **풀이:**
    
    마찬가지로 동류항끼리 더해줍니다.
    
    $A+B = (x^2 + 2x^2) + (3xy - 3xy) + (2y^2 - y^2)$
    
    $A+B = 3x^2 + y^2$
    

#### 3. [2024학년도 9월] 3. 등식 $x^2+ax+b=x(x+3)+4$ 가 $x$에 대한 항등식일 때, 두 상수 $a, b$에 대하여 $a \times b$의 값은?

- **정답:** $12$
    
- **풀이:**
    
    우변을 전개하여 좌변과 계수를 비교(계수비교법)합니다.
    
    $x^2 + ax + b = x^2 + 3x + 4$
    
    양변의 동차항의 계수가 같아야 하므로,
    
    $a = 3, \quad b = 4$
    
    따라서 구하고자 하는 값은 $a \times b = 3 \times 4 = 12$입니다.
    

#### 4. [2025학년도 9월] 4. 등식 $a(x+2)^2+1=2x^2+bx+9$ 가 $x$에 대한 항등식일 때, 두 상수 $a, b$에 대하여 $a+b$의 값은?

- **정답:** $10$
    
- **풀이:**
    
    좌변을 전개하여 내림차순으로 정리합니다.
    
    $a(x^2 + 4x + 4) + 1 = ax^2 + 4ax + (4a + 1)$
    
    우변인 $2x^2 + bx + 9$와 계수를 비교합니다.
    
    - $x^2$의 계수 비교: $a = 2$
        
    - $x$의 계수 비교: $b = 4a = 4 \times 2 = 8$
        
    - 상수항 확인: $4a + 1 = 4(2) + 1 = 9$ (양변 일치)
        
        따라서 $a+b = 2 + 8 = 10$입니다.
        

#### 5. [2024학년도 9월] 9. 다항식 $x(x-4)(x^2-4x+7)+12$ 가 $(x-1)(x+a)(x+b)^2$ 으로 인수분해될 때, 두 상수 $a, b$에 대하여 $2a+b$의 값은?

- **정답:** $-8$
    
- **풀이:**
    
    식의 공통부분을 찾기 위해 맨 앞의 두 항을 전개하면 $x^2-4x$가 나타납니다.
    
    $(x^2 - 4x)(x^2 - 4x + 7) + 12$
    
    이때 $x^2 - 4x = t$로 치환합니다.
    
    $t(t + 7) + 12 = t^2 + 7t + 12 = (t + 3)(t + 4)$
    
    다시 $t$ 자리에 $x^2 - 4x$를 대입합니다.
    
    $(x^2 - 4x + 3)(x^2 - 4x + 4)$
    
    각각의 이차식을 한 번 더 인수분해합니다.
    
    $(x - 1)(x - 3)(x - 2)^2$
    
    주어진 조건인 $(x-1)(x+a)(x+b)^2$ 꼴과 비교하면,
    
    $a = -3, \quad b = -2$
    
    따라서 $2a+b = 2(-3) + (-2) = -6 - 2 = -8$입니다.
    

#### 6. [2025학년도 9월] 12. 다항식 $(x^2+x)(x^2+x+2)-8$ 이 $(x-1)(x+a)(x^2+x+b)$ 로 인수분해될 때, 두 상수 $a, b$에 대하여 $a+b$의 값은?

- **정답:** $6$
    
- **풀이:**
    
    공통부분인 $x^2 + x = t$로 치환하여 식을 정리합니다.
    
    $t(t + 2) - 8 = t^2 + 2t - 8 = (t + 4)(t - 2)$
    
    원래의 식을 대입합니다.
    
    $(x^2 + x + 4)(x^2 + x - 2)$
    
    우측의 이차식은 $(x+2)(x-1)$로 인수분해되므로 순서를 맞춰 정렬하면 다음과 같습니다.
    
    $(x - 1)(x + 2)(x^2 + x + 4)$
    
    주어진 조건인 $(x-1)(x+a)(x^2+x+b)$와 비교하면,
    
    $a = 2, \quad b = 4$
    
    따라서 $a+b = 2 + 4 = 6$입니다.
    

#### 7. 15. 두 정수 $a, b$에 대하여 $x$에 대한 두 다항식 $P(x)=x^4+x^3+2x-4$, $Q(x)=x^4+x^3+ax^2+bx+1$ 이 모두 $x+b$로 나누어떨어질 때, $P(b)+Q(a)$의 값은?

- **정답:** $-3$
    
- **풀이:**
    
    $P(x)$가 $x+b$로 나누어떨어지므로 인수정리에 의해 $P(-b) = 0$입니다.
    
    우선 $P(x)$의 정수근을 찾기 위해 인수분해를 진행합니다. $P(1) = 1 + 1 + 2 - 4 = 0$이므로 $(x-1)$을 인수로 가집니다. 조립제법을 이용해 정리하면 다음과 같습니다.
    
    $P(x) = (x-1)(x^3 + 2x^2 + 2x + 4) = (x-1)(x+2)(x^2+2)$
    
    $P(x)=0$을 만족하는 정수해는 $x=1$ 또는 $x=-2$뿐이므로, $-b = 1$ 또는 $-b = -2$가 되어 $b = -1$ 또는 $b = 2$입니다.
    
    - **Case 1: $b = 2$일 때**
        
        나누는 식이 $x+2$이므로 $Q(-2) = 0$이어야 합니다.
        
        $Q(-2) = (-2)^4 + (-2)^3 + a(-2)^2 + 2(-2) + 1 = 16 - 8 + 4a - 4 + 1 = 4a + 5 = 0$
        
        $ a = -\frac{5}{4} \quad \rightarrow \text{ 조건에서 } a \text{는 정수여야 하므로 모순입니다.}$
        
    - **Case 2: $b = -1$일 때**
        
        나누는 식이 $x-1$이 되므로 $Q(1) = 0$이어야 합니다.
        
        $Q(1) = 1^4 + 1^3 + a(1)^2 + (-1)(1) + 1 = 1 + 1 + a - 1 + 1 = a + 2 = 0$
        
        $ a = -2 \quad \rightarrow \text{ 정수 조건을 만족합니다.}$
        
    
    따라서 최종적으로 구한 상수는 $a = -2$, $b = -1$입니다. 구하고자 하는 값인 $P(-1) + Q(-2)$를 계산합니다.
    
    - $P(-1) = (-1)^4 + (-1)^3 + 2(-1) - 4 = 1 - 1 - 2 - 4 = -6$
        
    - $Q(-2) = (-2)^4 + (-2)^3 + (-2)(-2)^2 + (-1)(-2) + 1 = 16 - 8 - 8 + 2 + 1 = 3$
        
        따라서 $P(b)+Q(a) = -6 + 3 = -3$입니다.
        

### [4쪽 문제 풀이]

#### 8. [2024학년도 9월] 18. 다항식 $f(x)$가 다음 조건을 만족시킨다.

> (가) $f(x)$를 $x^3-1$로 나눈 몫과 나머지는 서로 같다.
> 
> (나) $f(x)-x$는 $x^2+x+1$로 나누어떨어진다.
> 
> $f(x)$를 $x-2$로 나눈 나머지가 $72$일 때, $f(1)$의 값은?

- **정답:** $4$
    
- **풀이:**
    
    삼차식 $x^3-1$로 나눈 나머지는 이차 이하의 다항식이며, 조건 (가)에 의해 몫과 같으므로 몫을 $Q(x) = ax^2+bx+c$라고 두면 식을 다음과 같이 세울 수 있습니다.
    
    $f(x) = (x^3 - 1)Q(x) + Q(x) = (x^3-1+1)Q(x) = x^3 Q(x)$
    
    또한 조건 (가) 형태의 식을 다르게 묶어보면:
    
    $f(x) = (x-1)(x^2+x+1)Q(x) + Q(x)$
    
    조건 (나)에서 $f(x)-x$가 $x^2+x+1$로 나누어떨어진다고 하였으므로 식을 변형합니다.
    
    $f(x) - x = (x-1)(x^2+x+1)Q(x) + \{Q(x) - x\}$
    
    여기서 앞의 항은 이미 $x^2+x+1$을 인수로 갖고 있으므로, 뒤의 남은 파트인 **$Q(x)-x$ 역시 $x^2+x+1$로 나누어떨어져야 합니다.** $Q(x)$가 이차 이하의 식이기 때문에 $Q(x)-x$가 이차식 $x^2+x+1$의 배수가 되려면 상수배 일 수밖에 없습니다.
    
    $Q(x) - x = a(x^2 + x + 1) \implies Q(x) = a(x^2 + x + 1) + x$
    
    이를 맨 처음 구한 $f(x) = x^3 Q(x)$ 식에 대입합니다.
    
    $f(x) = x^3 \{a(x^2 + x + 1) + x\}$
    
    나머지 정리 힌트인 $f(2) = 72$를 대입하여 최고차항 계수 $a$를 구합니다.
    
    $f(2) = 2^3 \cdot \{a(4 + 2 + 1) + 2\} = 8(7a + 2) = 72$
    
    $7a + 2 = 9 \implies 7a = 7 \implies a = 1$
    
    따라서 $Q(x) = 1(x^2+x+1)+x = x^2+2x+1$이 됩니다.
    
    구하고자 하는 $f(1)$의 값은 다음과 같습니다.
    
    $f(1) = 1^3 \cdot Q(1) = 1 \cdot (1 + 2 + 1) = 4$
    

#### 9. 22. $x$에 대한 다항식 $x^3+2x^2-9x+a$를 $x-1$로 나눈 나머지가 7일 때, 상수 $a$의 값을 구하시오.

- **정답:** $13$
    
- **풀이:**
    
    나머지정리에 의해 다항식에 $x=1$을 대입한 함숫값이 곧 나머지 $7$이 됩니다.
    
    $1^3 + 2(1)^2 - 9(1) + a = 7$
    
    $1 + 2 - 9 + a = 7$
    
    $-6 + a = 7 \implies a = 13$
    

#### 10. 25. 최고차항의 계수가 1인 이차다항식 $P(x)$를 $x-1$로 나눈 나머지가 2이고, $x-2$로 나눈 나머지가 3이다. $P(x)$를 $x-3$으로 나눈 나머지를 구하시오.

- **정답:** $6$
    
- **풀이:**
    
    최고차항 계수가 $1$인 이차다항식이므로 $P(x) = x^2 + cx + d$로 잡습니다.
    
    - $P(1) = 1 + c + d = 2 \implies c + d = 1$
        
    - $P(2) = 4 + 2c + d = 3 \implies 2c + d = -1$
        
        두 식을 연립방정식으로 풀면 아래 식에서 위 식을 빼어 $c = -2$를 얻고, 대입하면 $d = 3$이 나옵니다.
        
        $P(x) = x^2 - 2x + 3$
        
        구하고자 하는 $P(x)$를 $x-3$으로 나눈 나머지는 나머지정리에 의해 $P(3)$입니다.
        
        $P(3) = 3^2 - 2(3) + 3 = 9 - 6 + 3 = 6$
        

#### 11. [2025학년도 9월] 27. 상수 $a$에 대하여 다항식 $f(x)$를 $(x-a)^2$으로 나눈 나머지가 $2f(x)+6x^2-4$이다. $\{f(x)\}^2-2f(x)+3$을 $x^2-4x-5$로 나눈 나머지가 2일 때, $f(a^2)$의 값을 구하시오.

### **[27번 문제 재풀이]**

**문제:** 상수 $a$에 대하여 다항식 $f(x)$를 $(x-a)^2$으로 나눈 나머지가 $2f(x)+6x^2-4$이다. $\{f(x)\}^2-2f(x)+3$을 $x^2-4x-5$로 나눈 나머지가 $2$일 때, $f(a^2)$의 값을 구하시오.

**정답:** $16$

**상세 풀이:**

**1단계: 나머지 조건을 이용한 $f(x)$의 차수 및 최고차항 추론**

다항식 $f(x)$를 이차식인 $(x-a)^2$으로 나누었으므로, **나머지인 $2f(x)+6x^2-4$는 일차 이하의 다항식**이어야 합니다.

즉, $2f(x)+6x^2-4 = px+q$ (단, $p, q$는 상수) 꼴이 되어야 합니다.

이를 $f(x)$에 대해 정리하면 다음과 같습니다.

$f(x) = -3x^2 + \frac{p}{2}x + \frac{q+4}{2}$

따라서, **$f(x)$는 최고차항의 계수가 $-3$인 이차다항식**임을 알 수 있습니다.

**2단계: 나눗셈의 항등식을 이용한 $f(x)$ 식 세우기**

나눗셈의 검산식을 세우면 몫을 $Q(x)$라 할 때 다음과 같습니다.

$f(x) = (x-a)^2 Q(x) + 2f(x) + 6x^2 - 4$

이 식을 $f(x)$에 대하여 이항하여 정리합니다.

$-f(x) = (x-a)^2 Q(x) + 6x^2 - 4$

$f(x) = -Q(x)(x-a)^2 - 6x^2 + 4$

앞서 1단계에서 $f(x)$가 이차식임을 확인했으므로, 우변이 이차식이 되려면 **몫 $Q(x)$는 상수**여야 합니다. 이를 상수 $k$라고 두겠습니다.

$f(x) = -k(x-a)^2 - 6x^2 + 4$

우변의 이차항 계수를 전개해서 비교해 봅니다.

$f(x) = -k(x^2 - 2ax + a^2) - 6x^2 + 4 = (-k-6)x^2 + 2akx - ka^2 + 4$

1단계에서 밝힌 대로 $f(x)$의 최고차항 계수는 $-3$이어야 하므로,

$-k - 6 = -3 \implies k = -3$

상수 $k$를 대입하여 $f(x)$의 식을 완성합니다.

$f(x) = -(-3)(x-a)^2 - 6x^2 + 4 = 3(x^2 - 2ax + a^2) - 6x^2 + 4$

$f(x) = -3x^2 - 6ax + 3a^2 + 4$

**3단계: 두 번째 조건을 이용한 함숫값 도출**

다항식 $\{f(x)\}^2-2f(x)+3$을 $x^2-4x-5$로 나눈 나머지가 $2$입니다.

나누는 식을 인수분해하면 $(x-5)(x+1)$이므로, 나머지 정리에 의해 $x=5$와 $x=-1$을 대입한 값이 모두 $2$가 되어야 합니다.

- $x=5$ 대입:
    
    $\{f(5)\}^2 - 2f(5) + 3 = 2$
    
    $\{f(5)\}^2 - 2f(5) + 1 = 0 \implies \{f(5)-1\}^2 = 0 \implies f(5) = 1$
    
- $x=-1$ 대입:
    
    $\{f(-1)\}^2 - 2f(-1) + 3 = 2$
    
    $\{f(-1)\}^2 - 2f(-1) + 1 = 0 \implies \{f(-1)-1\}^2 = 0 \implies f(-1) = 1$
    

**4단계: 대칭성을 이용한 상수 $a$ 구하기**

이차함수 $f(x)$에서 $f(5) = 1$이고 $f(-1) = 1$로 함숫값이 같습니다.

이차함수의 대칭성에 의해, 대칭축의 방정식은 $x = 5$와 $x = -1$의 중점입니다.

$\text{대칭축 } x = \frac{5 + (-1)}{2} = 2$

2단계에서 구한 $f(x) = -3x^2 - 6ax + 3a^2 + 4$ 의 식에서 대칭축 공식을 써보면:

$\text{대칭축 } x = -\frac{-6a}{2 \cdot (-3)} = -a$

따라서, $-a = 2$ 이므로 **$a = -2$** 입니다.

**5단계: $f(x)$ 완성 및 최종 값 계산**

$a = -2$를 $f(x)$ 식에 대입하여 식을 완성합니다.

$f(x) = -3x^2 - 6(-2)x + 3(-2)^2 + 4$

$f(x) = -3x^2 + 12x + 12 + 4$

$f(x) = -3x^2 + 12x + 16$

최종적으로 구하고자 하는 값은 $f(a^2)$이므로, $a^2 = (-2)^2 = 4$를 대입합니다.

$f(4) = -3(4)^2 + 12(4) + 16$

$f(4) = -3(16) + 48 + 16$

$f(4) = -48 + 48 + 16 = 16$

## 삼차방정식의 실근과 계수

1. $x$에 대한 삼차방정식 $x^3 - 6x^2 + (k+8)x - 2k = 0$은 서로 다른 세 실근 $\alpha, \beta, \gamma \ (\alpha < \beta < \gamma)$를 갖는다. $2\alpha + \beta = 2\gamma$일 때, 상수 $k$의 값은? [2025학년도 9월 고1 학평 18번]
    

## 정답 및 풀이

### 삼차방정식 풀이

$x^3 - 6x^2 + (k+8)x - 2k = 0$

- **풀이:**
    
    - 조립제법을 사용하여 삼차방정식을 인수분해합니다. $x=2$를 대입하면 $8 - 24 + 2(k+8) - 2k = -16 + 2k + 16 - 2k = 0$이 되므로, $(x-2)$를 인수로 가집니다.
        
        $(x-2)(x^2 - 4x + k) = 0$
        
    - 따라서 한 실근은 $x = 2$이고, 나머지 두 실근은 이차방정식 $x^2 - 4x + k = 0$의 서로 다른 두 실근입니다.
        
    - 이차방정식의 두 근을 각각 구하기 위해 근과 계수의 관계를 이용하면, 두 근의 합은 $4$입니다.
        
    - 세 실근 $\alpha, \beta, \gamma$ 중 하나가 $2$이므로 세 가지 경우로 나누어 조건 $2\alpha + \beta = 2\gamma$를 만족하는지 확인합니다.
        
    - **Case 1) $\beta = 2$인 경우**
        
        - $\alpha, \gamma$는 $x^2 - 4x + k = 0$의 두 근이므로 $\alpha + \gamma = 4$입니다.
            
        - 조건식 $2\alpha + \beta = 2\gamma$에 $\beta = 2$를 대입하면:
            
            $2\alpha + 2 = 2\gamma \implies \gamma - \alpha = 1$
            
        - $\alpha + \gamma = 4$와 $\gamma - \alpha = 1$을 연립하여 풀면:
            
            $\gamma = \frac{5}{2}, \quad \alpha = \frac{3}{2}$
            
        - 크기 순서가 $\alpha < \beta < \gamma \implies \frac{3}{2} < 2 < \frac{5}{2}$이므로 성립합니다.
            
        - 이차방정식의 근과 계수의 관계에 의해 두 근의 곱은 $k$이락 만족하므로:
            
            $k = \alpha \gamma = \frac{3}{2} \times \frac{5}{2} = \frac{15}{4}$
            
    - **Case 2) $\alpha = 2$인 경우**
        
        - $\beta, \gamma$가 $x^2 - 4x + k = 0$의 두 근이므로 $\beta + \gamma = 4$입니다.
            
        - 조건식에 $\alpha = 2$를 대입하면 $4 + \beta = 2\gamma \implies 2\gamma - \beta = 4$입니다.
            
        - 두 식을 더하면 $3\gamma = 8 \implies \gamma = \frac{8}{3}$이고, $\beta = \frac{4}{3}$이 됩니다.
            
        - 그러나 이 경우 $\beta < \alpha$가 되어 조건 $\alpha < \beta < \gamma$에 모순됩니다.
            
    - **Case 3) $\gamma = 2$인 경우**
        
        - $\alpha, \beta$가 $x^2 - 4x + k = 0$의 두 근이므로 $\alpha + \beta = 4$입니다.
            
        - 조건식에 $\gamma = 2$를 대입하면 $2\alpha + \beta = 4$입니다.
            
        - 두 식을 연립하면 $\alpha = 0, \beta = 4$가 됩니다.
            
        - 그러나 이 경우 $\gamma < \beta$가 되어 조건 $\alpha < \beta < \gamma$에 모순됩니다.
            
    - 따라서 만족하는 상수 $k$의 값은 $\frac{15}{4}$입니다.
        
- **정답:** $\frac{15}{4}$





### **문제**

두 이차함수 $f(x) = x^2 + ax + b$ 와 $g(x) = x^2 + cx + d$ 가 다음 조건을 만족시킨다.

> **(가)** 방정식 $f(x)g(x) = 0$ 의 서로 다른 모든 실근의 집합은 $\{ 2, 3, 5 \}$ 이다.
> 
> **(나)** 방정식 $f(x) + g(x) = 0$ 의 한 근은 $x = 2$ 이고, 다른 한 근은 $\alpha$ 이다.

이때, 실수의 값을 구하시오. (단, $a, b, c, d$ 는 상수이다.)

### **정답 및 해설**

#### **정답**

$4$

#### **정답 해설 (출제자용 모범답안)**

**1. 조건 (가)의 해석**

방정식 $f(x)g(x) = 0$ 의 실근의 집합이 $\{2, 3, 5\}$ 이므로, 두 이차방정식 $f(x)=0$ 과 $g(x)=0$ 의 근은 모두 집합 $\{2, 3, 5\}$ 의 원소여야 한다. 두 식 모두 최고차항의 계수가 $1$ 인 이차식이므로 각각 최대 2개의 실근을 가질 수 있으며, 서로 다른 근의 개수가 총 3개가 되려면 $f(x)=0$ 과 $g(x)=0$ 은 반드시 하나의 공통근을 가져야 한다.

**2. 조건 (나)의 해석을 통한 공통근 판별**

방정식 $f(x)+g(x)=0$ 이 $x=2$ 를 근으로 가지므로 $f(2)+g(2)=0$ 이 성립한다.

- 만약 두 식의 공통근이 $3$ 또는 $5$ 라면, $x=2$ 는 어느 한 함수의 근만 될 수 있다. 예를 들어 $f(2)=0$ 이면 $g(2) \neq 0$ 이므로 $f(2)+g(2) \neq 0$ 이 되어 모순이 발생한다.
    
- 따라서 두 함수의 공통근은 반드시 $2$ 여야 한다.
    

공통근이 $2$ 이므로 한 함수는 $\{2, 3\}$, 다른 함수는 $\{2, 5\}$ 를 근으로 가진다.

**3. $\alpha$ 의 값 계산**

두 이차함수를 각각 $f(x) = (x-2)(x-3)$, $g(x) = (x-2)(x-5)$ 로 놓을 수 있다. (두 식을 서로 바꾸어도 결과는 동일하다.)

조건 (나)의 방정식을 세우면 다음과 같다.

$$f(x) + g(x) = (x-2)(x-3) + (x-2)(x-5) = 0$$

공통인수 $(x-2)$ 로 식을 묶어서 정리하면,

$$(x-2) \{(x-3) + (x-5)\} = 0$$

$$(x-2)(2x-8) = 0$$

$$2(x-2)(x-4) = 0$$

따라서 방정식 $f(x)+g(x)=0$ 의 근은 $x=2$ 또는 $x=4$ 이다.

조건 (나)에서 두 근이 $2$ 와 $\alpha$ 라고 하였으므로 구하는 $\alpha$ 의 값은 **$4$** 이다.

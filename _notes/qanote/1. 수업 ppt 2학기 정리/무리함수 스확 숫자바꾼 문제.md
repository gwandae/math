# 무리함수 문제 답안지

---

## 문제1. 빈칸 채우기

**⑴** 무리함수 $y=\sqrt{a(x-p)}+q\ (a\neq 0)$의 그래프는 함수 $y=\sqrt{ax}$의 그래프를 $x$축 방향으로 **$p$**만큼, $y$축 방향으로 **$q$**만큼 평행이동한 것이다.

**⑵** 무리함수 $y=\sqrt{a(x-p)}+q\ (a\neq 0)$의 정의역은 $a>0$일 때 **${x \mid x \geq p}$**, $a<0$일 때 **${x \mid x \leq p}$**이고, 치역은 **${y \mid y \geq q}$**이다.

---

## 문제2. 그래프, 정의역, 치역

**⑴** $y=-\sqrt{3x-6}+4=-\sqrt{3(x-2)}+4$

$$\boxed{\text{정의역: } {x \mid x \geq 2},\quad \text{치역: } {y \mid y \leq 4}}$$

**풀이:**

- $3x-6 \geq 0$에서 $x \geq 2$ → 정의역
- $\sqrt{3(x-2)} \geq 0$이므로 $-\sqrt{3(x-2)} \leq 0$
- 따라서 $y = -\sqrt{3(x-2)}+4 \leq 4$ → 치역
- 시작점: $(2, 4)$, 아래로 뻗어나감

---

**⑵** $y=\sqrt{4-2x}-2=\sqrt{-2(x-2)}-2$

$$\boxed{\text{정의역: } {x \mid x \leq 2},\quad \text{치역: } {y \mid y \geq -2}}$$

**풀이:**

- $4-2x \geq 0$에서 $x \leq 2$ → 정의역
- $\sqrt{-2(x-2)} \geq 0$이므로 $y \geq -2$ → 치역
- 시작점: $(2, -2)$, 왼쪽 위로 뻗어나감

---

## 문제3. 평행이동과 점 통과

$$\boxed{p=-1}$$

**풀이:**

$y=\sqrt{3x}$를 $x$축 방향으로 $p$, $y$축 방향으로 $3$만큼 평행이동: $$y=\sqrt{3(x-p)}+3$$

점 $(2, 6)$을 지나므로: $$6=\sqrt{3(2-p)}+3$$ $$3=\sqrt{3(2-p)}$$ $$9=3(2-p)$$ $$3=2-p$$ $$p=-1$$

---

## 문제4. 최댓값과 최솟값

$$\boxed{\text{최댓값: } 7}$$

**풀이:**

$y=\sqrt{2x+a}+3$에서 $2\leq x\leq 6$

$y=\sqrt{2x+a}+3$은 증가함수이므로:

- $x=2$일 때 최솟값, $x=6$일 때 최댓값

최솟값이 $5$이므로: $$\sqrt{2 \cdot 2+a}+3=5$$ $$\sqrt{4+a}=2$$ $$4+a=4$$ $$a=0$$

따라서 $y=\sqrt{2x}+3$

최댓값 ($x=6$): $$y=\sqrt{2 \cdot 6}+3=\sqrt{12}+3=2\sqrt{3}+3$$

잠깐, 계산을 다시 확인하면: $$y=\sqrt{12}+3=2\sqrt{3}+3 \approx 6.46$$

정수로 나오게 하려면 $a$를 조정해야 할 수 있지만, 문제 그대로라면:

$$\boxed{\text{최댓값: } 2\sqrt{3}+3}$$

(또는 정수 답을 원하면: $a=2$로 설정 시, 최솟값 $\sqrt{6}+3$이 되어 맞지 않음)

**수정된 풀이 (정수 답이 나오도록):**

$a=0$일 때 검산:

- $x=2$: $y=\sqrt{4}+3=2+3=5$ ✓ (최솟값)
- $x=6$: $y=\sqrt{12}+3=2\sqrt{3}+3$

$$\therefore \boxed{\text{최댓값: } 2\sqrt{3}+3}$$

## 문제5. 상수 $a, b, c$ 구하기

$$\boxed{a=-3,\ b=9,\ c=-2}$$

**풀이:**

그래프에서 시작점 $(3, -2)$, 점 $(0, 1)$을 지남

$y=\sqrt{ax+b}+c$에서:

**① 시작점 $(3, -2)$:** 근호 안이 0 $$a \cdot 3+b=0 \quad \cdots (1)$$ $$c=-2$$

**② 점 $(0, 1)$ 통과:** $$1=\sqrt{a \cdot 0+b}+c=\sqrt{b}-2$$ $$\sqrt{b}=3$$ $$b=9$$

**③ (1)에서:** $$3a+9=0$$ $$a=-3$$

**검산:** $y=\sqrt{-3x+9}-2$

- $x=3$: $y=\sqrt{0}-2=-2$ ✓
- $x=0$: $y=\sqrt{9}-2=1$ ✓
- $a<0$이므로 왼쪽으로 뻗어나감 ✓

## 문제6. 스키드 마크 길이

$$\boxed{50\text{ m}}$$

**풀이:**

$f(x)=\sqrt{254kx}$, $k=0.5$, 속력 $5\sqrt{254}$ km/h

$$5\sqrt{254}=\sqrt{254 \times 0.5 \times x}$$ $$5\sqrt{254}=\sqrt{127x}$$

양변을 제곱: $$(5\sqrt{254})^2=127x$$ $$25 \times 254=127x$$ $$6350=127x$$ $$x=50$$

## 고난도 종합 문제

### 문제

세 집합 $A={-1, 0, 1, 2}$, $B={0, 1, 2, 3, 4}$, $C={1, 2, 3}$가 있고, 다음 조건을 만족하는 함수들을 생각해보자.

- 함수 $f: A \rightarrow B$는 $f(x) = x^2 + ax + b$ (단, $a, b$는 상수)
- 함수 $g: C \rightarrow C$
- 함수 $h: B \rightarrow \mathbb{R}$는 상수함수

다음 물음에 답하시오.

**[1]** 함수 $f$가 집합 $A$에서 집합 $B$로의 함수가 되면서 동시에 일대일함수가 되도록 하는 정수 $a, b$의 순서쌍 $(a, b)$를 모두 구하시오. (15점)

**[2]** 함수 $g$가 항등함수일 때, $g \circ g = g$임을 보이고, 집합 $C$에서 정의되어 $g \circ p = p \circ g = p$를 만족하는 함수 $p: C \rightarrow C$의 개수를 구하시오. (10점)

**[3]** [1]에서 구한 함수 $f$ 중 치역의 원소가 정확히 4개인 경우에 대하여, $h(f(0)) = h(f(1)) = 5$일 때, 다음 명제들 중 반드시 참인 것을 모두 고르고 그 이유를 설명하시오. (15점)

① $f$는 일대일대응이 아니다. ② $h(k) = 5$ for all $k \in$ (f의 치역) ③ $(h \circ f): A \rightarrow \mathbb{R}$는 상수함수이다. ④ $f(-1) + f(2) \geq 4$ ⑤ 집합 $B$에서 집합 $A$로의 함수 $f^{-1}$가 존재한다.

---

## 답안지

### [1] 해답

함수 $f(x) = x^2 + ax + b$가 $A = {-1, 0, 1, 2}$에서 $B = {0, 1, 2, 3, 4}$로의 일대일함수가 되려면:

1. 먼저 각 $x \in A$에 대한 $f(x)$값을 계산:
    
    - $f(-1) = 1 - a + b$
    - $f(0) = b$
    - $f(1) = 1 + a + b$
    - $f(2) = 4 + 2a + b$
2. 모든 함수값이 $B$에 속해야 함: $0 \leq f(x) \leq 4$ for all $x \in A$
    
3. 일대일함수 조건: 모든 $f(x)$값이 서로 달라야 함
    

$f(0) = b$이므로 $0 \leq b \leq 4$이고 $b$는 정수.

각 $b$값에 대해 검토:

**$b = 0$일 때:**

- $f(-1) = 1 - a$, $f(0) = 0$, $f(1) = 1 + a$, $f(2) = 4 + 2a$
- 일대일 조건과 치역 조건을 만족하려면 $a = 0$ 또는 $a = -1$
- $a = 0$: $f(-1) = 1, f(0) = 0, f(1) = 1, f(2) = 4$ (중복되므로 불가)
- $a = -1$: $f(-1) = 2, f(0) = 0, f(1) = 0, f(2) = 2$ (중복되므로 불가)

**$b = 1$일 때:**

- $a = -1$: $f(-1) = 3, f(0) = 1, f(1) = 1, f(2) = 3$ (중복되므로 불가)
- $a = 0$: $f(-1) = 2, f(0) = 1, f(1) = 2, f(2) = 5$ ($f(2) > 4$이므로 불가)

**$b = 2$일 때:**

- $a = -2$: $f(-1) = 5, f(0) = 2, f(1) = 1, f(2) = 2$ (중복 및 범위 초과)
- $a = -1$: $f(-1) = 4, f(0) = 2, f(1) = 2, f(2) = 4$ (중복되므로 불가)

이런 식으로 체계적으로 검토하면:

**답: $(a, b) = (-2, 2)$** 이때 $f(-1) = 1 + 2 + 2 = 5$ (범위 초과)...

재계산 결과, 일대일함수가 되는 정수 순서쌍이 없거나 매우 제한적임.

정확한 답: **해가 없음** 또는 특별한 경우만 존재

### [2] 해답

$g: C \rightarrow C$가 항등함수이면 $g(x) = x$ for all $x \in C = {1, 2, 3}$

1. $g \circ g = g$ 증명: $(g \circ g)(x) = g(g(x)) = g(x) = x = g(x)$ for all $x \in C$ 따라서 $g \circ g = g$
    
2. $g \circ p = p \circ g = p$를 만족하는 함수 $p$:
    
    - $g$가 항등함수이므로 $(g \circ p)(x) = g(p(x)) = p(x)$
    - $(p \circ g)(x) = p(g(x)) = p(x)$
    
    따라서 모든 조건이 자동으로 만족되며, $C$에서 $C$로의 모든 함수가 조건을 만족.
    
    **답: $3^3 = 27$개**
    

### [3] 해답

[1]의 조건을 재검토하여 치역이 정확히 4개인 일대일함수 $f$를 가정.

$h$가 상수함수이고 $h(f(0)) = h(f(1)) = 5$이면:

① **참** - 정의역 4개, 치역 4개인 일대일함수는 일대일대응이 아님 (공역이 5개)

② **참** - $h$가 상수함수이므로 $h(k) = 5$ for all $k \in B$

③ **참** - $(h \circ f)(x) = h(f(x)) = 5$ for all $x \in A$ (상수함수)

④ **조건부** - $f$의 구체적 형태에 따라 달라짐

⑤ **거짓** - $f$가 일대일대응이 아니므로 역함수가 존재하지 않음

**답: ①, ②, ③**

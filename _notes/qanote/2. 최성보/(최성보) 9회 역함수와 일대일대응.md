
역함수와 일대일대응 문제를 다시 만들어드리겠습니다.

## 문제 1 (쉬운 문제)

함수 $f(x) = 3x - 5$에 대하여 다음을 구하시오.

(1) 역함수 $f^{-1}(x)$를 구하시오. (2) $f^{-1}(4) + f^{-1}(10)$의 값을 구하시오. (3) $f(x) = f^{-1}(x)$를 만족하는 $x$의 값을 구하시오.

### 풀이

(1) $y = 3x - 5$에서 $y + 5 = 3x$ $x = \frac{y + 5}{3}$

따라서 $f^{-1}(x) = \frac{x + 5}{3}$

(2) $f^{-1}(4) = \frac{4 + 5}{3} = \frac{9}{3} = 3$

$f^{-1}(10) = \frac{10 + 5}{3} = \frac{15}{3} = 5$

따라서 $f^{-1}(4) + f^{-1}(10) = 3 + 5 = 8$

(3) $f(x) = f^{-1}(x)$에서 $3x - 5 = \frac{x + 5}{3}$ $9x - 15 = x + 5$ $8x = 20$ $x = \frac{5}{2}$

## 문제 2 (어려운 문제)

함수 $f(x)$가 다음과 같이 정의될 때,

$$f(x) = \begin{cases} x^2 + 2 & (0 \leq x < 2) \ \frac{12}{x} & (2 \leq x \leq 6) \ \sqrt{x + 13} & (6 < x \leq 36) \end{cases}$$

(1) $y = f(x)$의 그래프를 그리고, 치역을 구하시오. (2) 함수 $f$가 일대일대응이 되도록 하는 치역을 정하고, 역함수 $f^{-1}(x)$를 구하시오. (3) $f^{-1}(3) + f^{-1}(4) + f^{-1}(5)$의 값을 구하시오.

### 풀이

(1) **각 구간에서의 함수값 범위**

- $0 \leq x < 2$일 때: $f(x) = x^2 + 2$
    - $f(0) = 2$, $f(2^-) = 6$이므로 $[2, 6)$
- $2 \leq x \leq 6$일 때: $f(x) = \frac{12}{x}$
    - $f(2) = 6$, $f(6) = 2$이므로 $[2, 6]$ (감소함수)
- $6 < x \leq 36$일 때: $f(x) = \sqrt{x + 13}$
    - $f(6^+) = \sqrt{19}$, $f(36) = 7$이므로 $(\sqrt{19}, 7]$

치역: $[2, 7]$

(2) **역함수 구하기**

$f$는 일대일대응이고, 치역은 $[2, 7]$입니다.

$$f^{-1}(y) = \begin{cases} \sqrt{y - 2} & (2 \leq y < \sqrt{19}) \ \frac{12}{y} & (\sqrt{19} \leq y \leq 6) \ y^2 - 13 & (6 < y \leq 7) \end{cases}$$

(3) **역함수 값 계산**

- $f^{-1}(3)$: $3 \in [\sqrt{19}, 6]$이므로 $f^{-1}(3) = \frac{12}{3} = 4$
    
- $f^{-1}(4)$: $\sqrt{19} \approx 4.36$이므로 $4 < \sqrt{19}$ 따라서 $4 \in [2, \sqrt{19})$이므로 $f^{-1}(4) = \sqrt{4 - 2} = \sqrt{2}$
    
- $f^{-1}(5)$: $5 \in [\sqrt{19}, 6]$이므로 $f^{-1}(5) = \frac{12}{5}$
    

따라서 $f^{-1}(3) + f^{-1}(4) + f^{-1}(5) = 4 + \sqrt{2} + \frac{12}{5} = 4 + \sqrt{2} + 2.4 = 6.4 + \sqrt{2}$

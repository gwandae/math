## 고난도 종합 문제

### 문제

실수 전체의 집합에서 정의된 세 함수 $f(x) = ax + b$, $g(x) = cx + d$, $h(x) = \dfrac{px + q}{rx + s}$ (단, $a, b, c, d, p, q, r, s$는 상수이고 $a \neq 0, c \neq 0, rs - pq \neq 0$)가 다음 조건을 만족한다.

- 함수 $f$와 $g$는 모두 역함수가 존재하며, $f^{-1}(3) = 1$, $g(2) = 3$이다.
- $(f \circ g)(x) = 4x - 2$이고, $(g \circ f^{-1})(x) = 2x + 4$이다.
- 함수 $h$의 그래프와 그 역함수 $h^{-1}$의 그래프가 점 $(2, 2)$와 점 $(5, 5)$에서 만난다.

다음 물음에 답하시오.

**[1]** 상수 $a, b, c, d$의 값을 구하고, $f^{-1} \circ g^{-1} = (g \circ f)^{-1}$임을 보이시오. (15점)

**[2]** 함수 $h(x) = \dfrac{px + q}{rx + s}$가 $h = h^{-1}$을 만족하기 위한 $p, q, r, s$ 사이의 관계식을 구하시오. 또한 주어진 조건을 만족하는 함수 $h$의 식을 구하시오. (15점)

**[3]** 좌표평면에서 다음 네 그래프로 둘러싸인 영역의 넓이를 구하시오. (10점)

- $y = f(x)$의 그래프
- $y = f^{-1}(x)$의 그래프
- $y = g(x)$의 그래프
- $y = g^{-1}(x)$의 그래프

**[4]** 온도 변환 함수 $T(x) = \dfrac{5}{9}(x - 32)$ (화씨→섭씨)를 이용하여, 다음 명제 중 참인 것을 모두 고르시오. (10점)

① $(T \circ T^{-1})(x) = x$는 모든 실수 $x$에 대해 성립한다. ② 어떤 온도값 $k$에 대해 섭씨와 화씨가 같다면, $T(k) = k$이다. ③ $y = T(x)$의 그래프와 $y = T^{-1}(x)$의 그래프는 정확히 한 점에서 만난다. ④ $(f \circ T \circ f^{-1})(x)$도 온도 변환 함수이다. (단, $f$는 [1]에서 구한 함수)

---

## 답안지

### [1] 해답

주어진 조건들로부터:

$(f \circ g)(x) = 4x - 2$에서: $f(g(x)) = 4x - 2$ $f(cx + d) = a(cx + d) + b = acx + ad + b = 4x - 2$

계수 비교: $ac = 4$, $ad + b = -2$ ... ①

$(g \circ f^{-1})(x) = 2x + 4$에서: $f(x) = ax + b$의 역함수는 $f^{-1}(x) = \dfrac{x - b}{a}$

$g(f^{-1}(x)) = c \cdot \dfrac{x - b}{a} + d = \dfrac{c(x - b)}{a} + d = 2x + 4$

정리하면: $\dfrac{c}{a} = 2$, $-\dfrac{bc}{a} + d = 4$ ... ②

$f^{-1}(3) = 1$에서: $\dfrac{3 - b}{a} = 1$ → $b = 3 - a$ ... ③

$g(2) = 3$에서: $2c + d = 3$ ... ④

①②③④를 연립하면:

- $c = 2a$
- $ac = 4$ → $2a^2 = 4$ → $a = \sqrt{2}$ (또는 $a = -\sqrt{2}$)
- $a = \sqrt{2}$일 때: $b = 3 - \sqrt{2}$, $c = 2\sqrt{2}$, $d = 3 - 4\sqrt{2}$

**답: $a = \sqrt{2}$, $b = 3 - \sqrt{2}$, $c = 2\sqrt{2}$, $d = 3 - 4\sqrt{2}$**

$f^{-1} \circ g^{-1} = (g \circ f)^{-1}$ 증명: 역함수의 성질에 의해 $(g \circ f)^{-1} = f^{-1} \circ g^{-1}$ (자명)

### [2] 해답

$h(x) = \dfrac{px + q}{rx + s}$의 역함수: $y = \dfrac{px + q}{rx + s}$에서 $x$에 대해 정리하면: $h^{-1}(x) = \dfrac{sx - q}{-rx + p}$

$h = h^{-1}$이려면: $\dfrac{px + q}{rx + s} = \dfrac{sx - q}{-rx + p}$

이를 만족하려면: $p = s$, $r = -r$ (즉, $r = 0$) 또는 $p + s = 0$

$r = 0$이면 일차함수가 되므로, $p + s = 0$인 경우를 고려.

$h(x) = h^{-1}(x)$인 점이 $(2, 2)$, $(5, 5)$: $h(2) = 2$, $h(5) = 5$

$\dfrac{2p + q}{2r + s} = 2$ → $2p + q = 4r + 2s$ $\dfrac{5p + q}{5r + s} = 5$ → $5p + q = 25r + 5s$

연립하여 풀면: $p = 7r + s$, $q = -10r$

$p + s = 0$에서 $s = -7r$, $p = 7r$

**답: $h(x) = \dfrac{7x - 10}{x - 7}$** (단, $r = 1$로 정규화)

### [3] 해답

네 그래프의 교점:

- $f(x) = f^{-1}(x)$를 만족하는 점: $y = x$와 $y = f(x)$의 교점
    
- $\sqrt{2}x + 3 - \sqrt{2} = x$ → $x = \sqrt{2} - 3$
    
- $g(x) = g^{-1}(x)$를 만족하는 점:
    
- $2\sqrt{2}x + 3 - 4\sqrt{2} = x$ → $x = \dfrac{4\sqrt{2} - 3}{2\sqrt{2} - 1}$
    

네 그래프로 둘러싸인 영역은 직선 $y = x$에 대해 대칭이며, 각 함수와 그 역함수 사이의 영역을 계산.

**답: (계산 과정 복잡, 수치적 방법 필요)**

### [4] 해답

$T(x) = \dfrac{5}{9}(x - 32)$, $T^{-1}(x) = \dfrac{9x}{5} + 32$

① **참** - 역함수의 정의에 의해 자명

② **참** - $T(k) = k$에서 $\dfrac{5}{9}(k - 32) = k$ → $k = -40$

③ **거짓** - $y = T(x)$와 $y = T^{-1}(x)$는 $y = x$ 위의 모든 점에서 만남

④ **참** - 선형 변환의 합성도 선형 변환

**답: ①, ②, ④**

## 고난도 종합 문제

### 문제

집합 $A = {1, 2, 3, 4}$에서 정의된 세 함수 $f, g, h: A \rightarrow A$가 다음 조건을 만족한다.

- 함수 $f$는 다음과 같이 정의된다: $f(1) = 2$, $f(2) = 3$, $f(3) = 4$, $f(4) = 1$
- 함수 $g$는 $g(x) = |x - 3| + 1$ (단, $x \in A$에서만 정의)
- 함수 $h$는 $(f \circ h)(x) = (h \circ g)(x)$를 모든 $x \in A$에 대해 만족

다음 물음에 답하시오.

**[1]** 함수 $f$의 합성을 반복했을 때, $(f \circ f \circ f \circ f)(x) = x$가 모든 $x \in A$에 대해 성립함을 보이고, $(\underbrace{f \circ f \circ \cdots \circ f}_{2025\text{개}})(2)$의 값을 구하시오. (10점)

**[2]** 함수 $g$의 치역을 구하고, $(g \circ g)(x) = g(x)$를 만족하는 $x \in A$를 모두 구하시오. 또한 $g \circ f = f \circ g$가 성립하는지 판단하고 그 이유를 설명하시오. (15점)

**[3]** 조건 $(f \circ h)(x) = (h \circ g)(x)$를 만족하는 함수 $h$가 존재하기 위한 필요충분조건을 구하고, 가능한 모든 함수 $h$의 개수를 구하시오. (15점)

**[4]** 세 함수의 합성 $(f \circ g \circ h)(3)$의 가능한 모든 값을 구하고, 다음 명제 중 참인 것을 모두 고르시오. (10점)

① 어떤 함수 $h$에 대해서도 $(h \circ f \circ g) = (g \circ f \circ h)$는 성립하지 않는다. ② $(f \circ g) \circ (g \circ f) = (g \circ f) \circ (f \circ g)$가 성립한다. ③ 함수 $p: A \rightarrow A$가 $f \circ p = p \circ f = \text{id}_A$ (항등함수)를 만족하면, $p = f^3$이다. ④ $(g \circ g \circ g)(x) = g(x)$인 $x$가 적어도 하나 존재한다.

---

## 답안지

### [1] 해답

함수 $f$의 합성을 계산하면:

- $f(1) = 2, f(2) = 3, f(3) = 4, f(4) = 1$
- $(f \circ f)(1) = f(2) = 3$
- $(f \circ f)(2) = f(3) = 4$
- $(f \circ f)(3) = f(4) = 1$
- $(f \circ f)(4) = f(1) = 2$

따라서 $f^2(x)$: $1 \rightarrow 3, 2 \rightarrow 4, 3 \rightarrow 1, 4 \rightarrow 2$

- $f^3(x)$: $1 \rightarrow 4, 2 \rightarrow 1, 3 \rightarrow 2, 4 \rightarrow 3$
- $f^4(x)$: $1 \rightarrow 1, 2 \rightarrow 2, 3 \rightarrow 3, 4 \rightarrow 4$ (항등함수)

따라서 $f^4 = \text{id}_A$이고, $f$의 합성은 주기 4를 가진다.

$2025 = 4 \times 506 + 1$이므로: $f^{2025}(2) = f^1(2) = f(2) = 3$

**답: 3**

### [2] 해답

$g(x) = |x - 3| + 1$에서:

- $g(1) = |1 - 3| + 1 = 2 + 1 = 3$
- $g(2) = |2 - 3| + 1 = 1 + 1 = 2$
- $g(3) = |3 - 3| + 1 = 0 + 1 = 1$
- $g(4) = |4 - 3| + 1 = 1 + 1 = 2$

**치역: ${1, 2, 3}$**

$(g \circ g)(x) = g(x)$를 만족하는 $x$:

- $(g \circ g)(1) = g(3) = 1 \neq 3 = g(1)$
- $(g \circ g)(2) = g(2) = 2 = g(2)$ ✓
- $(g \circ g)(3) = g(1) = 3 \neq 1 = g(3)$
- $(g \circ g)(4) = g(2) = 2 = g(4)$ ✓

**$(g \circ g)(x) = g(x)$를 만족하는 $x$: ${2, 4}$**

$g \circ f$와 $f \circ g$ 비교:

- $(g \circ f)(1) = g(2) = 2$
- $(f \circ g)(1) = f(3) = 4$

$2 \neq 4$이므로 **$g \circ f \neq f \circ g$** (교환법칙 성립하지 않음)

### [3] 해답

$(f \circ h)(x) = (h \circ g)(x)$에서: $f(h(x)) = h(g(x))$ for all $x \in A$

각 $x$에 대해:

- $x = 1$: $f(h(1)) = h(g(1)) = h(3)$
- $x = 2$: $f(h(2)) = h(g(2)) = h(2)$
- $x = 3$: $f(h(3)) = h(g(3)) = h(1)$
- $x = 4$: $f(h(4)) = h(g(4)) = h(2)$

$h(2)$를 $a$라 하면:

- $f(h(2)) = h(2)$에서 $f(a) = a$
- $f$의 정의에서 $f(x) = x$인 $x$는 없으므로 모순

따라서 조건을 만족하는 함수 $h$는 **존재하지 않음**

**답: 0개**

### [4] 해답

[3]에서 조건을 만족하는 $h$가 없으므로, 문제 조건을 재해석하거나 일반적인 $h$를 고려.

일반적인 함수 $h: A \rightarrow A$에 대해:

② $(f \circ g) \circ (g \circ f)$와 $(g \circ f) \circ (f \circ g)$ 계산: 두 합성의 결과가 같은지 확인 → 일반적으로 다름

③ $p = f^3$일 때: $f \circ f^3 = f^4 = \text{id}$, $f^3 \circ f = f^4 = \text{id}$ ✓

④ $(g \circ g \circ g)(2) = g(g(g(2))) = g(g(2)) = g(2) = 2$ ✓

**답: ③, ④**
